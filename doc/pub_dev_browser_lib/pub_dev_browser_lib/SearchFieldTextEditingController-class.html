SearchFieldTextEditingController

SearchFieldTextEditingController class

A TextEditingController for the search field.

This controller provides syntax highlighting for the search field qualifiers and query text. The qualifiers are colored differently from the query text to provide a visual distinction between the two. The qualifiers are also styled differently from each other.
