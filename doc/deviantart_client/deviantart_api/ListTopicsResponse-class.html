ListTopicsResponse class

Response for listing topics.

The ListTopicsResponse class has a constructor that takes a required parameter topics.

The ListTopicsResponse.fromJSON factory constructor creates an instance of ListTopicsResponse from a JSON decoded list. It maps each decoded topic to an instance of m.Topic using the m.Topic.fromJSON method, and then converts the mapped topics to a list.
